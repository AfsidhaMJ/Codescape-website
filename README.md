# CODESCAPE – Digital Presence Website

## Overview
This project is a modern, high-performance, multi-page website designed for CODESCAPE as part of CAPEATHON.  
The goal is to represent CODESCAPE’s engineering excellence, innovation, and reliability through strong UX, clean UI, and optimized performance.
